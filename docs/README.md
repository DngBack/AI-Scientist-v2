# Tài liệu dự án AI Scientist v2

Thư mục này chứa tài liệu mô tả tính năng và đánh giá điểm mạnh/điểm yếu của codebase.

## Nội dung

| File | Nội dung |
|------|----------|
| **[FEATURES.md](FEATURES.md)** | Giải thích chi tiết các tính năng: ideation, BFTS (4 stage), aggregate plots, writeup, review, và luồng dữ liệu. |
| **[STRENGTHS_AND_WEAKNESSES.md](STRENGTHS_AND_WEAKNESSES.md)** | Điểm mạnh và điểm yếu của hệ thống (kiến trúc, bảo mật, chi phí, phụ thuộc, bảo trì, v.v.). |

## Đọc nhanh

- Muốn biết **hệ thống đang làm gì, có những tính năng gì** → đọc **FEATURES.md**.
- Muốn biết **ưu/nhược điểm khi dùng hoặc phát triển** → đọc **STRENGTHS_AND_WEAKNESSES.md**.

Tài liệu được tạo từ việc đọc và phân tích code trong repo (không thay thế README gốc ở root).
